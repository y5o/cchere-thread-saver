The tool is used to download the entire thread on online Chinese forum ccthere.com and save them to the hard disk with as many files as the number of pages. Then you can pick a user to put all his posts together, sort them by date, and merge them into a new html file.
Another script can extract all the email addresses in the thread.

用来下载西西河某个thread的某个用户的所有文章，按页存成文件。并且可以这个讨论串中将某个作者的所有文章提取出来，并且按时间排序存成一个新的文件。另一个脚本可以提取出这个thread中所有的email地址。
